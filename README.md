# api_final
## API для учебного проекта yatube Яндекс Практикум


api_final это проект для взаимодействия с помощью архитектуры REST API с сайтом yatube

## Features

- Получение публикации
- Создание публикации
- Обновление публикации
- Частичное обновление публикации
- Удаление публикации
- Получение комментариев
- Добавление комментария
- Обновление комментария
- Частичное обновление комментария
- Удаление комментария
- Список сообществ
- Информация о сообществе
- Подписки
- Подписка
- Получить JWT-токен
- Обновить JWT-токен
- Проверить JWT-токен

## Installation

Для работы api_final требуется python 3.9+

Для запуска потребуется

```sh
Django==3.2.16
pytest==6.2.4
pytest-pythonpath==0.7.3
pytest-django==4.4.0
djangorestframework==3.12.4
djangorestframework-simplejwt==4.7.2
Pillow==9.3.0
PyJWT==2.1.0
requests==2.26.0
djoser==2.1.0
```
Или установить зависимости из requirements.txt
